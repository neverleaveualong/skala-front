# SKALA-FRONT

본 저장소는 **SKALA 과정**의 HTML, CSS, JavaScript 전 과정을 통합 수행한 프론트엔드 포털 프로젝트입니다. 
웹 표준 시맨틱 구조 설계부터 성능 최적화 레이아웃, 모바일 반응형 대응, 그리고 비동기 모듈러 아키텍처까지 프론트엔드 실무 역량 강화를 목표로 작성되었습니다.

---

## 🛠️ 개발 환경 및 실행 방법
본 프로젝트는 **ES6 JavaScript Module (`type="module"`)**을 사용하므로 CORS 정책 준수를 위해 로컬 개발 서버가 필요합니다.

*   **VS Code 사용 시**: `Live Server` 또는 `Five Server` 확장을 설치하여 `html/index.html`을 오픈합니다.
*   **CLI 환경 사용 시**:
    ```bash
    npx live-server html/index.html
    ```
