추가로 학습한 HTML, CSS, JavaScript

  • HTML 전역 속성 실습: contenteditable을 사용한 실시간 메모장 영역 구현
  • 고급 텍스트 포맷팅: <del>(취소선), <ins>(밑줄), <small>, <i> 등의 다양한 태그 활용
  • 리스트 구조 응용: 다차원 중첩 리스트 구조(ul inside ul) 설계 및 순서 리스트의 type="A" 속성 사용
  • 북마크 링크 이동: <a href="#id">를 이용한 페이지 내 특정 위치(오전/오후 일과) 바로가기 구현
  • 유니코드 이모지 적용: 이모지를 텍스트가 아닌 유니코드 10진수 기호(&#128564; 등)로 작성하는 기술 학습
  • HTML 폼 심화 실습: 다양한 폼 요소(fieldset, legend, datalist, range, color 등)와 입력 제어 속성(readonly, required, autofocus) 활용
  • 멀티미디어 태그 활용: <audio>, <video>, <source>, <img> 요소를 이용한 여행지 소개 미디어 재생 구현
  • HTML5 시맨틱 레이아웃: <header>, <nav>, <main>, <aside> 등을 도입하여 포털 형태의 웹 문서 구조 설계
  • CSS3 고급 선택자 제어: 부모 존재 감지(:has), 특정 조건 제외(:not), 의사 클래스(:hover, :focus, :active)의 유기적 설계
  • 모던 CSS 박스 모델 응용: 테두리 병합(border-collapse) 및 테이블 셀 병합(rowspan) 렌더링 오류를 개선한 시간표 그리드 설계
  • 부드러운 호버 트랜지션: 마우스 올렸을 때 배경색 등이 서서히 바뀌도록 transition 설정
  • JavaScript 기본 동작 및 연동: prompt()와 while(true) 루프/break를 활용한 숫자 맞추기(Up-Down) 게임, for 반복문을 활용한 과목 평균 성적 계산기, JavaScript Object와 array를 순회하여 출력하는 가방 속 소지품 연동 실습
